Design a modern mobile/web app UI called **Orbit**, a student event discovery platform for UT Dallas that helps students find networking events, internships, career fairs, club meetings, and company visits using Nebula Labs campus data.

**Design Style & Theme**

The entire UI should follow a **space / galaxy aesthetic**. The interface should feel like navigating a universe of opportunities.

Visual style:

* Dark space background with deep navy, black, and cosmic gradients
* Aurora-inspired accent colors (purple, cyan, blue, pink)
* Stars twinkling subtly in the background
* Constellation-style UI connections
* Glowing UI elements on hover
* Smooth zoom transitions like traveling through space

Micro-interactions:

* ⭐ Stars twinkle randomly in the background
* 🌠 Shooting star animation appears when a **new event notification** appears with the message *“New event in your orbit”*
* ✨ Buttons glow softly on hover
* Smooth **zoom-in animation** when selecting events (like zooming into a star system)

Typography should feel **modern, futuristic, but clean and readable**.

AI Assistant

### Screen 5 — Discover (Star Map View)

This screen should look like a **galaxy map or star chart**.

Each **star represents a type of opportunity**:

⭐ Internships
⭐ Career Fairs
⭐ Company Visits
⭐ Club Events
⭐ Tech Talks
⭐ Workshops

Users can **pan and zoom around the galaxy**.

Clicking a star:

* The camera **zooms into it like a solar system**

Click **Internships star → shows internship related events**

### Overall UX Principles

* Smooth transitions between screens (like traveling through space)
* Glow effects on interactive elements
* Subtle star animations to make the interface feel alive
* Minimal but elegant UI
* Futuristic but intuitive navigation

This design should feel like **exploring a galaxy of opportunities**, where every event is a star waiting to be discovered.
